# 子网计算器 v1.0.5 Release Notes

本版本是 **代码审查优化版**：对全项目做了一次系统体检（27 个问题），把 19 项实际隐患 + 5 项工程化改进合并发布。所有用户可见的改动向后兼容。

## 🐛 修复

- **IP 类型识别扩展**：新增 RFC 5737 文档示例地址（`192.0.2.0/24`、`198.51.100.0/24`、`203.0.113.0/24`）和 RFC 6598 CGNAT（`100.64.0.0/10`），不再被错判为公网
- **IP 类型按网络地址判断**：`192.168.1.5 /16` 现在正确显示"私有地址"（之前按 IP 本身会被判为公网 C 类）
- **/31 /32 提示**：这两种边界情况现在带说明文案（RFC 3021 点对点 / 单主机地址）
- **Toast 提前消失 bug**：原来 `setTimeout(1800)` 与动画 2100ms 不同步，toast 在 fadeOut 中途被截断。改用 `animationend` 监听最后一帧
- **CI release body 失效根因修复**：`release` job 之前缺 `actions/checkout` 导致 `body_path` 文件永远读不到。本版本起 release job 自带 checkout
- **a11y 引用**：`<input aria-describedby="ipInput-hint">` 现在指向真实存在的 `#ipInput-hint` 元素（之前是空引用）
- **Electron 加载失败白屏**：`did-fail-load` 现在弹 `dialog.showErrorBox` 提示错误码和原因
- **debounce + change 竞态**：mask 下拉 change 事件会先 clear debounce 计时器，避免 400ms 后重复计算
- **默认 IP 占位**：input 移除了硬编码的 `value="192.168.1.1"`，避免新用户误以为这是历史记录

## ✨ 改进

- **前端代码分文件**：`www/index.html`（UI/样式）拆出 `www/subnet.js`（纯函数库）+ `www/app.js`（DOM/事件），为单测铺路
- **CSP 收紧**：`<script src>` 全部外链，去掉 `script-src 'unsafe-inline'`
- **移动端主题色**：`<meta name="theme-color" content="#0078d4">` 让 Android 状态栏匹配品牌色
- **前端 IP 类型徽章标签从"IP类型"改为"网络类型"**，语义更准确（按网络地址判断）

## 🛠️ 工程化

- **Vitest 单测**：55 个 case 覆盖所有 IP/掩码/CIDR 转换与 IP 类型识别（`npm test`）
- **ESLint 9**：flat config 风格，包含 no-undef / eqeqeq / prefer-const 等基础规则（`npm run lint`）
- **`engines` 字段**：声明 `node >= 20, npm >= 9`，避免低版本装依赖时出问题
- **7za postinstall 真正工作**：`package.json` 加 `postinstall` 钩子调用 `build/7za-wrapper/build.js`，Windows 上自动编译 7za shim（之前 README 撒谎说 postinstall 会编译，实际 build.js 不存在）
- **Program.cs 兼容老 csc.exe**：用 `Arguments` 字符串 + 手工 quote 替代 `ProcessStartInfo.ArgumentList`（后者需要 .NET Core 2.1+）
- **Electron 包排除测试代码**：`build.files` 加上 `!**/tests/**` 和 `!**/eslint.config.*`

## 📚 文档

- 删除虚假的"iOS (Capacitor, source only)"声明
- 7za 章节：写清楚实际工作流（postinstall → build.js → 编译 C# shim）
