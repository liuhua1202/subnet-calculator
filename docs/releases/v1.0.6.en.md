# Subnet Calculator v1.0.6 Release Notes

This release is an **icon + UI tweak**: brand-new SVG icon plus two user-requested UI changes. Fully backward-compatible — no breaking changes.

## 🎨 Icon

- **Brand-new SVG icon**: replaced the old `🌐` emoji + simple geometric pattern with a modern calculator silhouette featuring a `display: "1010"` numeric screen
- **All 15 Android icons regenerated**: `mipmap-{mdpi,hdpi,xhdpi,xxhdpi,xxxhdpi}/ic_launcher.png` + `ic_launcher_round.png` + `ic_launcher_foreground.png` (adaptive icon foreground, 432×432, 72dp safe area)
- **Web header uses inline SVG**: renders natively in browser / desktop WebView / Capacitor WebView — same look everywhere
- **Icon source**: `www/icon.svg` (128×128, `currentColor`, freely resizable via `sharp`)
- **Re-runnable generator**: `build/generate-icons.js` (prereq `npm install --no-save sharp`)

## ✨ UI adjustments

- **Restored default IP placeholder `192.168.1.1`**: v1.0.5 removed the hardcoded default to avoid misleading "history" perception, but new users opening the app to an empty input didn't know what to type. Reverting to the default-value form based on real feedback
- **Removed the "Total Hosts" tile**: "Usable Hosts" already conveys the count; one less tile gives the layout more breathing room. Removed directly from the `tiles` array in `www/app.js`
