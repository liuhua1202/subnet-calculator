# 子网计算器 v1.0.4 Release Notes

本次发布以 bugfix 为主，所有改动向后兼容。

## 🐛 修复

- **IP 类型识别**：`getIpType` 新增 `0.0.0.0/8` 保留段判断，`0.1.2.3` 等保留地址不再误判为公网
- **二进制高亮溢出**：`renderBinRow` 仅对整 octet 全为网络位的字节加 `.highlight` 蓝底；`/25`、`/13` 等非 8 位对齐 CIDR 不再显示溢出
- **CI versionCode 碰撞**：`versionCode` 计算改 `%03d%03d`，`1.0.10` 与 `1.1.0` 不再生成相同 `versionCode`（避免 Google Play 上传冲突）

## ✨ 改进

- **UI**：新增 `.ip-type-badge.broadcast` / `.reserved` 样式，`255.255.255.255` 与保留地址 badge 不再裸奔
- **docs**：修正 `main.js` `minWidth` 注释（与 CSS 实际行为对齐）
- **cleanup**：删除 `renderBinRow` 中无用的 `totalBits` 变量
