# 子网计算器 v1.0.6 Release Notes

本版本是 **图标与 UI 微调版**：替换全新 SVG 图标 + 两处用户反馈的 UI 调整。功能上完全向后兼容，升级后无破坏性变化。

## 🎨 图标

- **全新 SVG 图标**：原 `🌐` emoji + 简单几何图样替换为带 `display: "1010"` 数字屏幕的现代计算器造型
- **15 个 Android 图标全部更新**：`mipmap-{mdpi,hdpi,xhdpi,xxhdpi,xxxhdpi}/ic_launcher.png` + `ic_launcher_round.png` + `ic_launcher_foreground.png`（adaptive icon 前景，432×432，72dp 安全区）
- **Web header 内联 SVG**：浏览器内实时显示，desktop / webview / Capacitor WebView 一致
- **图标源文件**：`www/icon.svg`（128×128，`currentColor`，可被 `sharp` 任意缩放）
- **可重跑生成脚本**：`build/generate-icons.js`（前置 `npm install --no-save sharp`）

## ✨ UI 调整

- **恢复默认 IP 占位 `192.168.1.1`**：v1.0.5 移除了硬编码默认值（避免被误认成"历史记录"），但实际反馈是新用户打开应用面对空 input 不知道该输什么。这次回退到带默认值的形态
- **删除"总主机数" tile**：保留"可用主机数"已够用，省一个 tile 给呼吸感。`www/app.js` 的 `tiles` 数组直接删一行
