# Subnet Calculator v1.0.4 Release Notes

This release is a bugfix batch. All changes are backward-compatible.

## 🐛 Fixes

- **IP type detection**: `getIpType` now recognises `0.0.0.0/8` as reserved — addresses like `0.1.2.3` no longer mis-classified as public
- **Binary highlight overflow**: `renderBinRow` only applies the `.highlight` blue background to octets entirely covered by network bits; partial octets (e.g. `/25`, `/13`) no longer misleading
- **CI versionCode collision**: padding switched to `%03d%03d` so `1.0.10` and `1.1.0` no longer produce identical `versionCode` (prevents Google Play upload conflicts)

## ✨ Improvements

- **UI**: added `.ip-type-badge.broadcast` / `.reserved` styles — `255.255.255.255` and reserved addresses now display a styled badge
- **docs**: corrected `main.js` `minWidth` comment to match actual CSS behaviour
- **cleanup**: removed unused `totalBits` variable in `renderBinRow`
